Detailed Description

Motion planner class interface.

Remarks
Child classes should be implemented with update_gscene and plan_algorithm. Child classes' constructor should call MotionInterface.__init__(self, pscene). Specify NAME = "" for a child class as a class variable To use online planning, additionally implement init_online_algorithm, step_online_plan, update_online and update_target_joint.
