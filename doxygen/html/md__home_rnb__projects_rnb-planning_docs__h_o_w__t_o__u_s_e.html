HOW TO

Documents

  • The main instruction document is rnb-planning/ReadMe.md
  • Other instruction documents are stored in rnb-planning/docs

Doxygen

  • Doxygen documentation can be browsed in rnb-planning/doxygen/html/index.html
  • To host doxygen documents, run following command (port=5000 by default) ``` cd "$RNB_PLANNING_DIR"doxygen && python ./host_doxygen.py ```
  • To update doxygen document, launch doxywizard and load rnb-planning/doxygen/Doxyfile
